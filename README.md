# Virtual Memory Management Simulator

An interactive web-based educational tool that visualizes Operating System memory management concepts and page replacement algorithms in real time.

## Overview

Virtual Memory Management Simulator helps students understand how Operating Systems manage memory using different page replacement algorithms. The simulator provides real-time visualization, performance analysis, and algorithm comparison through an intuitive user interface.

## Features

* Real-time page replacement simulation
* FIFO (First In First Out)
* LRU (Least Recently Used)
* Optimal Page Replacement
* LFU (Least Frequently Used)
* Step-by-step frame visualization
* Page Hit and Page Fault tracking
* Hit Ratio and Fault Ratio calculation
* Algorithm comparison dashboard
* Interactive charts using Chart.js
* Learning Hub for Operating System concepts
* Dark and Light mode support
* Recent simulation history

## How It Works

1. Enter the number of memory frames.
2. Enter a page reference string.
3. Select a page replacement algorithm.
4. Click **Run Simulation**.
5. Observe frame allocation and page replacement in real time.
6. Analyze page hits, page faults, hit ratio, and fault ratio.
7. Compare all algorithms using the comparison dashboard.
