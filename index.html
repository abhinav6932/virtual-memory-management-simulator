Learning Hub

Virtual Memory

A memory management technique that creates an illusion of a very large memory, providing an idealized abstraction of storage resources.

Pages & Frames

Page: Logical memory is divided into blocks of the same size.
Frame: Physical memory is divided into fixed-sized blocks.

Page Fault

Occurs when a program attempts to access a block of memory not currently in physical memory, requiring the OS to fetch it from secondary storage.

FIFO Algorithm

First In First Out: The simplest page-replacement algorithm. When a page must be replaced, the oldest page is chosen.

LRU Algorithm

Least Recently Used: Replaces the page that has not been used for the longest period of time.

Optimal Algorithm

Optimal Page Replacement: Replaces the page that will not be used for the longest time in the future. Has the lowest fault rate but is impossible in practice.

LFU Algorithm

Least Frequently Used: Replaces the page with the smallest count of accesses, based on frequency.

Why Page Replacement Matters

Page replacement algorithms help operating systems efficiently utilize limited physical memory by deciding which pages should be replaced when memory becomes full.
